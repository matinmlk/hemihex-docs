# HemiHex Tutorial: Introduction to the Jetson Orin Nano

*An introductory guide for AI-powered inspection and automation
systems.*

## Overview

![Jetson Orin Nano Core Board](/img/docs/2023050500001.png)

The **Jetson Orin Nano** is NVIDIA's compact yet powerful module
designed for edge-AI applications---ideal for HemiHex's automated
inspection, computer vision, and robotics systems.

------------------------------------------------------------------------

## Core Specifications

-   **CPU:** ARM Cortex‑A78AE, 6 cores, up to 1.5 GHz\
-   **GPU:** Ampere GA10B, 1024 CUDA cores, 32 Tensor Cores, 625 MHz\
-   **AI Compute:** Up to 1.28 TFLOPs FP32 / 40 TOPS INT8\
-   **Video:** Hardware decoder only, CPU‑assisted encoding up to
    1080p30\
-   **Power:** Efficient 15W module TDP

------------------------------------------------------------------------

## Orin Nano vs. Jetson Nano

![Performance Comparison](/img/docs/2023050500002.png)

The Orin Nano delivers an **80× AI performance improvement** over the
classic Jetson Nano.\
This leap enables real‑time multi‑camera inspection, advanced OCR, and
high‑accuracy defect detection suitable for industrial environments.

------------------------------------------------------------------------

## Orin Nano 4GB vs 8GB

![4GB vs 8GB Comparison](/img/docs/2023050500003.png)

  Feature    Orin Nano 4GB        Orin Nano 8GB
  ---------- -------------------- -----------------------------
  Memory     4GB LPDDR5           8GB LPDDR5

Choose the **8GB** version for AI inspection devices, defect detection,
and automated multi‑sensor systems.
