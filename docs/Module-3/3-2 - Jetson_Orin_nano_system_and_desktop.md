# Jetson Orin Nano System and Desktop Overview

## Jetson Orin Nano System

This section details the system installed on the Jetson Orin Nano, which is a Yahboom-specific version built upon the official Jetson system, running Ubuntu 20.04. The comprehensive system configuration is illustrated below:

![](/img/docs/3-2/2023050500001.png)

Key specifications and pre-installed software components include:

*   **Operating System:** Linux 64-bit (Ubuntu 20.04)
*   **Python Version:** Python 3.8
*   **CUDA:** 11.4
*   **cuDNN:** 8.6
*   **TensorRT:** 8.5
*   **OpenCV:** 4.5.4
*   **JetPack Version:** 5.1.1
*   **Hardware:** Jetson Orin Nano (16GB)

The default system password is: `yahboom`

## Desktop Interface Overview

The Linux desktop environment provided is designed for simplicity and ease of use. Upon successful startup, users are automatically directed to the desktop. By connecting standard peripherals such as a mouse, keyboard, and monitor, you can interact with and operate the Jetson Orin Nano in the same manner as a conventional computer.
