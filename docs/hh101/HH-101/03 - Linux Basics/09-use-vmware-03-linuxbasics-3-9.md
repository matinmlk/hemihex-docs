# Use VMware

## 1. VMware download

Official website: https://www.broadcom.com/

:::note
: You need to register an account to download the software. Currently, VMware Workstation Pro is free for personal use!
:::

```text
The VMware Workstation Pro software package will be provided in the software data folder
```

Go to the official website and select "VMware Products":

![image-20241229125957135](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229125957135.png)

Select "Desktop Hypervisor":

![image-20241229130123821](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229130123821.png)

Click "DOWNLOAD FUSION OR WORKSTATION" to enter the download interface:

![image-20241229130644998](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229130644998-1735448805324-2.png)

Search for "VMware Workstation" on the download interface Pro‚Äù:

![image-20241229130949039](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229130949039.png)

Select the corresponding platform to download: We choose Windows here

![image-20241229131056721](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229131056721.png)

Select the version number to download:

![image-20241229131144579](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229131144579.png)

Download software:

![image-20241229131320847](/img/docs/jetson/03-LinuxBasics/3-9/image-20241229131320847.png)
